============================================================
              ALL MATCHUPS (ALL REGIONS)
============================================================

This mod expands Dynamic Campaign / Conquest matchups so that
every region allows every faction pairing between:

  ger, rus, fin, usa, eng

Both directions are included (e.g., "ger rus" and "rus ger").

------------------------------------------------------------
WHAT THIS MOD CHANGES
------------------------------------------------------------

Overrides:
  set/dynamic_campaign/values.set

Edits:
  Regions -> <Region> -> AvailableMatchups

------------------------------------------------------------
INSTALLATION
------------------------------------------------------------

Place the folder here:
  <game_root>/mods/

Enable the mod in-game:
  Main Menu -> Options -> Mods -> Enable

Restart the game if prompted.

------------------------------------------------------------
NOTES / COMPATIBILITY
------------------------------------------------------------

- If your install/DLC does not fully support a faction in a given
  region, some matchups may appear but could behave unexpectedly.
- Will conflict with other mods that also override:
    set/dynamic_campaign/values.set

------------------------------------------------------------
VERSION
------------------------------------------------------------

Designed for game version: 1.059
minGameVersion: 1.050
maxGameVersion: 1.080
